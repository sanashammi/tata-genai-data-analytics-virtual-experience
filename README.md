# Tata GenAI-Powered Data Analytics Virtual Experience

## Project Overview
This project was completed as part of the Tata GenAI-Powered Data Analytics Virtual Experience Program on Forage.

## Key Tasks Completed
- Exploratory Data Analysis (EDA)
- Missing Value Analysis
- Delinquency Risk Assessment
- Predictive Modeling Framework
- Business Recommendations
- AI-Powered Collections Strategy

## Key Insights
- Missed payments are a strong predictor of delinquency.
- High credit utilization increases financial risk.
- Customers with high debt-to-income ratios require proactive intervention.
